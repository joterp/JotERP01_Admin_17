Procedure to use this feature
Step 1 : Click on SELECT FILES to select the template file to be imported
Step 2 : Click on IMPORT button. The name of the file will display in the drop-down IMPORTED FILES
Step 3 : Select relevant items
Step 4 : Select ASSIGNED TRAITS on the extreme-left
Step 5 : Click blue-color button ASSIGN TRAITS TO SELECTED ITEMS, at the bottom of the grid
Step 6 : Edit each item as required, and mark CRITICAL ITEMS as required
Step 7 : Once all data is ready, click on button SAVE SELECTED ITEMS TO LIVE DATABASE, at the bottom of the grid
